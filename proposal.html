Introduction

We are creating a 2D game where the character will automatically run but you ultimately have control of which direction the character will go. The player will have to escape from multiple enemies and will have to collect points to go onto the next level. Some of the other graphics will be taken from games.

    Functional Requirements

  1. Menu (start, options, quit)
  2. Multiple Levels (different themes)
  3. Different characters
  4. Music and sounds

    Non-Functional Requirements

  1. not CPU heavy
  2. program written in Java

System Evolution

Will support upcoming versions of Java.
